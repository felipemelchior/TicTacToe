TicTacToe Minimax

Este código tem como objetivo simular uma jogada no jogo da velha, utilizando minimax e pode alpha e beta.
Resultado ainda está ruim

Uso
Dependências:
$ python3

$ biblioteca Numpy - Instale com -> sudo pip3 install numpy 

Para rodar o código use: 

$ python3 main.py <dimensao M> <dimensao N> <jogador 1-2> <condição de vitoria> <estado>

Após rodar, o código irá ler os dados e a jogada automaticamente! <br>
Após finalizado os cálculos, por padrão, será impresso na tela as coordenadas que a IA jogará
